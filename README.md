# California National Parks Data Analysis

This project analyzes visitation patterns across California National Parks to understand why some parks receive more visitors than others.

The analysis examines relationships between park area, location, and budget to explore factors influencing park popularity.

## Tools Used
- Wolfram Mathematica
- Data Visualization
- Data Analysis

## Dataset
Dataset includes park visitation statistics, park area, and budget information.

## Project Goal
Use data analysis to understand how park size, location, and funding affect visitor numbers.

## Files
- Dataset (.csv)
- Mathematica analysis notebook
